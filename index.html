What Courses We Offer

Animation

We provide a basic animation course which helps our team learn how to animate objects within a Three Dimensionsal space.

This allows them to rig and animate anything,
ranging from
characters to spaceships

Character Creation

Using applications such as Blender and Z Brush, we train our team to sculpt characters, be it realistic or stylised characters.

This could range from human characters to monster-type beings, all directed by the story of the game.

3D Modelling

A basic understanding on how to create 3D Objects such as cars, houses etc which will be used as assets within games.

C# Code

We provide a full basic coures on learning how to use C# in order to create the mechanics for the game, such as interacting with objects.

Since we primarily use is Unity's Game Engine, it is important for our developers to learn C# since its the code used in Unity.

Game Theory

Game Theory involves anything from Character Story to Level Design.

Its Important for an aspiring Game Developer to learn Game Theory because it allows the player to create a proper flow from the beginning of the game until the end and provide full immersion.
